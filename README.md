# E-Commerce Platform | Professional Portfolio

Portafolio interactivo.
Desarrollo de plataforma de comercio electrónico completa. MERN stack.

## Demo

**[Ver Portafolio](https://fdevg04.github.io/rs-ecommerce-portfolio/)**

## Descripción

Este portafolio presenta el proceso de desarrollo, arquitectura y aspectos técnicos de un proyecto e-commerce en producción. Incluye diagramas interactivos, documentación del stack tecnológico y análisis de soluciones.

## Stack del Portafolio

- **Framework:** React 19 + Vite
- **Estilos:** Tailwind CSS
- **Despliegue:** GitHub Pages
- **CI/CD:** GitHub Actions

## Secciones

- Descripción del proyecto
- Stack tecnológico detallado
- Funcionalidades implementadas
- Arquitectura del sistema (diagrama interactivo)
- Desafíos técnicos y soluciones
- Métricas del proyecto

## Desarrollo Local

```bash
# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# Construir para producción
npm run build
```
